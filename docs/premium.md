<h1 align="center">Premium</h1>

AgeVerifier's core age-verification system is **free** — welcoming members, date-of-birth verification, role assignment,
lobby cleanup, logging, and more all work at no cost. **Premium** unlocks a set of advanced features for servers that
want extra security and automation.

<p align="center"><strong><a href="https://strykerdevelopment.com">Get premium on the dashboard →</a></strong></p>

---

## What premium unlocks

| Feature                                  | What it does                                                                                                                                                                                                                                                                      |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Verification Page**                    | Your own fully customizable web verification page — upload images, set colors, add an intro, and display your rules in style. Part of Website verification.                                                                                                                       |
| **Automated lobby cleanup & kicking**    | Automatically remove (and optionally kick) members who sit unverified in the lobby, using *Clean Lobby Days* and *Kick on Cleanup*.                                                                                                                                               |
| **Reverification**                       | Ask existing members to verify again to access restricted areas (e.g. NSFW), including the reverification roles and the reverify log channel. See the [Reverify reference](commands/Reverify.html).                                                                               |
| **Exit Survey**                          | Automatically DM departing members a feedback survey.                                                                                                                                                                                                                             |
| **Leave Message**                        | Post a notification to a channel whenever a member leaves the server.                                                                                                                                                                                                             |
| **Join Requirements (gatekeeping)**      | Screen new members as they join — account age, avatar, mutual servers, bot detection, ban records, and online-presence checks — and automatically log or kick those who fail. See [Configuration → Join Requirements](configuration.html#join-requirements-gatekeeping--premium). |
| **Advanced verification modes**          | ID Verification, Basic + ID, and Website verification. (Basic date-of-birth verification is free.)                                                                                                                                                                                |
| **Manual ID requests** (`/idcheck send`) | Send a member a direct request to provide ID for a manual check.                                                                                                                                                                                                                  |

The verification methods themselves are compared on the [Verification Methods](verification-methods.html) page.

> AgeVerifier's premium lineup keeps growing — you can even **suggest your own premium features**, with more coming
> soon.

## Why do premium features exist?

AgeVerifier is free to use, and running it isn't. Premium exists to keep the bot healthy and sustainable for everyone:

- **It covers the bot's hosting costs** — keeping AgeVerifier online, fast, and available around the clock.
- **Some commands and features use more resources** than others, and premium helps offset that heavier load.

In short, premium keeps the free core running well for every server while funding the more demanding features for those who want them.

---

## How to get premium

Premium is purchased through the **web dashboard**:

1. Go to the [dashboard](https://strykerdevelopment.com) and sign in with Discord.
2. Select the server you want to upgrade.
3. Choose premium and follow the checkout steps. Current plans and pricing are shown there.

Premium applies **per server**, and it's active for the duration of your subscription. Once it's active, the premium
features above become available immediately — no re-inviting the bot required.

> **Tip:** you can confirm a server's status at any time — premium-only commands will tell you if premium is required,
> and your settings on the dashboard reflect what's unlocked.

## Frequently asked

**Is the free version limited or trial-based?**
No. The core verification system is fully functional for free, with no time limit. Premium is purely additive.

**What happens when premium expires?**
Premium-only features stop working, but your core setup and all your other settings keep running exactly as before.
Re-subscribing re-enables them.

**Do my members need premium?**
No — premium is bought by the server, and it affects the whole server. Members never pay anything.
